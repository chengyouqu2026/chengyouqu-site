核心框架

不可能三角：副业的三个要素——收入稳定、上手快、收入高——很难同时成立。理解这个规律，能帮你做出更理性的选择。

三条路径：

  • 收入高 + 上手快 = 不稳定（信息差套利）
  • 稳定 + 上手快 = 收入低（勤快/便宜）
  • 稳定 + 收入高 = 上手慢（专业门槛）

四象限定位：根据「时间多少」×「专业度高低」，判断你适合哪种副业。

  • 时间多 + 技能成长中 → 执行型选手，先跑起来
  • 时间多 + 专业度高 → 全能型选手，走高端路线
  • 时间紧 + 专业度高 → 产品化选手，做睡后收入
  • 时间碎片 + 技能成长中 → 轻启动选手，从小事做起

利弊穿透：每个方向都有优势与风险，系统不会只说好话。适合你的路会说清楚为什么适合，不适合你的路也会直说——节省你的试错成本。

探索模式：如果你还没确定做什么副业，选"还在探索中"，系统会帮你制定2周探索计划，推荐低成本试错方向，告诉你怎么判断一个方向适不适合自己。

使用建议
  • 看到任何副业项目，先用本工具分析是否适合自己
  • 重点关注「利弊分析」和「不推荐方向」——少踩坑比多赚钱更重要
  • 不同项目多测几次，对比找到最优方向
  • 分析结果会保存，随时回顾和调整
